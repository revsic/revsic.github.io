올해는 회사 생활을 정리하고, 대학원에서 한 해를 보냈다.

**2월**\
**졸업**

{{< figure src="/images/post/on2025/graduate.jpeg" width="80%" caption="25.02.21.졸업식" >}}

휴학 기간이 길었지만, 결국 7년 간의 학교 생활이 끝났다. \
고졸로 회사 생활을 시작해 대졸에 다 달으니 감회가 새롭다.

원래는 졸업식에 가지 않을 계획이었다. \
퇴사 전 인수인계와 업무 정리로 바쁜 시기를 보내고 있었고, \
친한 동기들은 이미 졸업하여 덩그러니 남은 졸업식에 기대감을 가지지 않았다.

그렇게 회사에 있다 졸업 프로젝트를 같이 한 동기의 연락을 받았다. \
그래도 마지막인데 가서 사진이라도 하나 찍어야 하지 않겠냐 말이다.

부모님께는 안 가도 된다 이야기해 놓고, \
막상 안 가려니 미련이 남아 오후 반차를 내고 졸업식에 갔다.

졸업식 분위기는 밝고 희망에 차 보였다. \
이제는 더 오지 않을 학교에 괜한 아쉬움이 남기도 했다. \
별 행사나 일정 없이, 친구와 사진 하나 찍고 집으로 돌아왔다. 

돌아다니던 중 연락이 끊겨 이젠 어색해진 동기들과 인사도 했다. \
많이들 가고 싶던 회사에 취업하여 밝은 표정을 하고 있었다. \
괜히 부럽기도 했고, 나는 6년 동안 회사에 다니면서 무얼 해왔나 괜한 생각도 들었다. 

졸업이라고 당장 달라지는 건 없었다. \
내일도 출근할 것이고, 머지않아 퇴사 후 대학원으로 간다. \
번듯한 대기업 직장인이 된 친구들을 보니 괜히 대학원이 더 초라해 보이기도 했다. \
지금까지의 커리어는 괜찮았나 생각도 하고, 창업, 중소기업, 학업까지 해봤으니 \
다음은 대기업도 한 번 가보고 싶단 생각도 들었다.

정상성이라는 게 무섭다. \
때맞춰 대학 가서 취업에 결혼까지, \
나는 썩 정상적이진 않은 20대를 보내왔나 싶기도 했다. 

괜한 비교를 뒤로하고 마지막으로 학교에서 내려왔다.

---

**이사**

독립한 후로 가능한 직장 30분 거리 내에서 집을 구하고 있다. \
6시에 퇴근 후 집 와서 밥 먹고 뭐라도 하려면, 이보다 더 멀어지고 싶진 않다.

벌써 세 번째 이사다. \
이번엔 대학원 근처로 가려한다. 

세 번의 자취방이 모두 대학가였다 보니, 부동산에 갈 시기를 잘 골라야 했다. \
12월 초에 가면 아직 방이 안 나왔고, 1월 설 지나고 가면 신입생들이 다 계약했다. \
12월 말에 가면 2주만 있다 오라고 돌려보낸다.

이번에는 실패하지 않겠다, 눈치를 열심히 보다 1월 초 부동산에 갔다. \
제시한 기준은 2층 이상 남향, 4평 이상이었다.

동네가 산에 붙어 있다 보니, 전반적으로 언덕이 많았다. \
언덕이 많으니 건물이 남향이어도 앞 건물에 채광이 막혔다. \
채광을 챙기고 나니 집이 오래됐거나 컨디션이 썩 좋지 않은 경우도 잦았다.

돌고 돌던 중, 낮 시간 채광이 만족스러운 집을 하나 찾았고, \
컨디션도 나빠 보이지 않아 바로 계약했다. 

처음 집을 보았을 때는 꽤 넓어 보였는데, \
입주를 하고 나니 직전 집의 절반 정도 크기였다. \
그중에 괜찮은 방이었는데도, 대학원생 월급으로 갈 수 있는 곳이 몇 없구나 싶었다.

그럼에도 여름까지는 지금 집에 굉장히 만족하고 지냈다. \
나름 아늑하고 지낼만했다. \
문제는 겨울이었다.

입주 당시 도배를 새로 했길래 집주인이 친절하구나 싶었다. \
알고 보니 외벽 단열이 거의 안 되어, 겨울의 시작과 함께 벽지에 곰팡이가 올라왔다. \
꽤나 심한 수준이어서, 나 나갈 때 도배 다시 하겠구나 싶었다.

곰팡이 문제를 넘어 난방비도 꽤 나온다. \
새삼 누워 있는데 벽에서 한기가 들어오는 집은 또 처음이다.

남향 채광을 찾다 보니 보통 방이 외벽에 붙어 있는데, \
원룸 건물의 단열이 고만고만한지, 지금까지 자취방 모두에서 곰팡이에 시달리긴 했다. \
그럼에도 이렇게 차원이 다른 곰팡이는 마음이 아프다.

또 다른 문제는 언덕이다. \
단순 올라가는 것이 힘든 게 문제가 아니었다.

새벽에 눈이 온 날, 언덕을 내려가는데 자꾸 쿵쿵 소리가 들렸다. \
소리가 나는 곳에서는 사람이 한 명씩 넘어져 있었다. \
눈이 녹다 얼어 언덕은 빙판이 되어 있었고, \
나 또한 넘어짐을 피할 수 없었다. 

문제는 집에서 내려가는 모든 언덕이 각도가 꽤 있어, \
내려가는 것만이 문제가 아닌, 올라가는 것 또한 굉장히 아슬아슬했다.

다음 자취방은 언덕도 완만하고, 단열도 잘 되는 남향집을 찾을 것이다. \
돈 많이 벌어야겠다.

**3월**\
**입학**

{{< figure src="/images/post/on2025/snowy.jpeg" width="80%" caption="25.03.18.학교" >}}

연구실은 2월 중순부터 출근하였다. \
퇴사 후 거진 바로 출근하게 되었다. 

특이하게 올해는 3월에 눈이 왔는데, 학교에 쌓인 눈을 보며 감탄했다. \
그와 별개로 학교 언덕도 눈이 녹다 얼어, 아슬아슬한 출퇴근이 이어졌다. \
그 외에는 나름 설레는 첫 한 달을 보낸 것 같다.

입학식은 따로 가지 않았고, 나중에 부모님과 정문에서 사진 정도만 찍었다. \
부모님은 내가 대학원에 가길 바라셨는데, \
학교 정문에서 더 좋아하시는 모습을 보니 그럭저럭 나쁘지 않은 선택이었나 싶었다.

학교에 처음 와서 느낀 점은 두 가지였는데, \
하나는 정말 학교가 넓다는 것이다.

집 앞 마을버스 종점에서 버스를 타고 학교에 가면, \
연구실 건물은 반대 종점 앞에 있어 대략 30분 정도 이동 시간이 걸린다. \
분명 학교에서 가장 가까운 전철역인데, 버스로 30분이라는 것이 첫 충격이었다. \
30분이면 본가에서 학부 학교까지 자전거를 타도 갈 수 있는 거리였다.

두 번째는 첫 연구실 방문 당시였다. \
연구실은 건축된 지 1년 채 안 된 신축 건물에 있었는데, 건물이 꽤 독특하게 생겼다. \
정육면체 콘크리트 건물이 즐비하던 학부 때와는 또 달랐다. 

문제는 그런 건물들은 일관되게 1층을 적당히 돌면 출입구가 나오는데, \
여기는 문을 열고 들어가면 어디 테라스로 나오고, \
문을 열고 나가면 갑자기 주차장이 나오는 알 수 없는 구조로 되어 있었다. 

나는 첫날부터 건물에 들어가기 위해 30분을 내둘렀고, \
건물에서 나오기 위해 또 30분을 내둘렀다. \
이 정도면 길치라 그런 건가 싶을 정도였다. 

---

첫 연구실 방문 후 교수님과 어떤 것들을 연구하고 싶은지 면담을 하였다. \
30분 정도 일찍 출발하여 망정이지, 첫날부터 출입구 찾다가 지각할 뻔했다.

연구실은 본래 3D 비전을 주제로 삼고 있었다. \
그리고 근래에는 생성 모델과 로봇 연구로의 확장을 고려하던 상황이었다. \
나는 생성 모델의 연구를 주제로 삼고자 하였고, 그리던 그림에 관하여 이야기드렸다. 

먼저 물리적 요인을 잘 반영하지 못하던 기존 생성 모델의 문제점을 이야기드리고, \
이에 대한 개선을 첫 주제로 이야기드렸다. \
꽤 긍정적 반응을 보이셨고, 문제점에 대해 더 구체화해 보기로 했다. 

둘째는 생성 모델의 이론 파트를 연구하고 싶다 이야기드렸다. \
대학원에 지원한 것은 돈이 안 되는 연구에 관한 갈증을 포함하고 있었다. \
회사 연구팀에 있으면서 돈이 되는 연구를 수행하는데 초점을 맞추어 왔고, \
그에 대한 피로도를 낮추는 시간을 가지고 싶었다.

당시는 Consistency Model에 관심을 가지고 있었고, \
이에 대한 이론적 해석이나 수렴성 등을 해석하고 개선하는 것도 괜찮을 것 같았다. \
이 부분도 긍정적 반응을 보이셨고, 기존 연구의 재생산으로 시작해 보기로 했다. 

교수님께서는 연구 주제에 열려 계셨고, \
오히려 단체의 미션과 어울리는 연구 스토리라인을 만드는 것은 늘 해오던 일이었기에, \
큰 무리 없이 주제에 관하여 설득할 수 있지 않았나 싶기도 하다.

{{< figure src="/images/post/on2025/spring.png" width="80%" caption="25.04.10.학교" >}}

연구실 출근 후 맞닥뜨린 첫 어려움은 연구 보단 학교 생활이었다. \
특히 수강신청에서 애를 많이 먹았다.

학부 때는 장바구니 같은 일정 없이 모든 과목이 선착순이었다. \
여기는 시스템도 다르고 졸업 요건도 다르니, 수강신청 당일 뭘 들어야 하는지, \
어떻게 신청해야 하는지 굉장히 얼타고 있었다. 

그런 모습이 안쓰러워 보였는지, 선배들이 와 수강신청을 도와주었다. \
뭘 들어야 하는지, 뭐가 좀 듣기 편한지, 어떻게 신청하는지 새로운 것투성이다. \
다른 회사로 이직을 했어도 이렇게 새롭진 않았을 것 같았다. 

첫 학기는 지적 허영을 채워도 되지 않을까 싶어, 재밌어 보이는 과목을 몇 집었다. \
보안 회사에 다니면서 관심 있게 살피던 프로그램 정적 분석과 \
확률 모델의 이론 연구를 하고 싶었다 보니 확률 모델 과목을 신청했다. \
그리고 이는 굉장한 실책이었다. 

과목 자체는 굉장히 흥미롭고 재밌는 주제를 다뤘다. \
문제는 두 과목 모두 과제가 많기로 유명한 과목이었다. \
선배들의 만류에는 이유가 있었다.

특히 한 과목은 매주 쪽지 시험, 매주 강의 요약문 제출, \
6개의 문제 풀이 과제와 기말 대체 팀 프로젝트까지 \
학교가 다른 대학원에 비해 코스웍 양이 많다고는 들었지만, 예상을 넘은 분량이었다. \
연구 주제 잡으랴 과제하랴 1학기는 어떻게 지나갔는지 기억도 잘 나지 않는다.

그나마 팀 프로젝트가 정도가 기억에 남는다. \
학부 때는 졸업 프로젝트를 제외하면 팀 프로젝트가 과제인 경우는 없었고, \
몇몇 있던 것도 알고 지내던 동기와 함께하여 큰 무리가 없었다. 

금번의 팀 프로젝트는 온전히 모르는 이를 모집하여 진행해야 했다. \
다른 팀원의 모집 공고에 컨택하여 팀을 구성하고, \
프로젝트를 진행함에 있어 모든 경험이 새로웠다. 

특히 프로젝트 결과를 논문 형태의 글로 적어내야 했는데, \
Grammarly와 GPT가 없었다면 정말 시간이 많이 들었겠구나 싶었다. \
지금 시대에 대학원에 온 것이 다행인가 싶기도 했다.

다행히 흔히 말하는 빌런이 우리 팀에 있지는 않았고, 꽤 무난히 마무리 지었다.

**4월**\
**영어 공부**

학교에 와서 처음으로 부담스러웠던 것은 주제 설정도 무엇도 아닌 \
세미나를 영어로 진행해야 한다는 것이었다.

연구실에서는 1주일에 한 번 한 사람이 원하는 주제로 세미나를 진행하는데, \
신입생인 나는 5월에 첫 세미나가 배정되었다. \
세미나는 발표부터 질의응답까지 모든 과정이 영어로 진행되었고, \
이는 꽤 부담으로 느껴졌다.

생각해 보면 고등학생 때도 영어 공부를 그렇게 열심히 하진 않았던 것 같다. \
크게 공부 시간을 많이 할애하지 않아도, \
적당히 보고 적당히 찍으면 답을 맞히는 것에는 큰 무리가 없었다. \
논문을 볼 때에도 단어군이 고만고만하니 읽고 이해하는데 큰 무리가 없던 것 같다.

문제는 인생에 영어로 대화한 경험은 손에 꼽는다는 것이다. \
직전 회사가 미국에 본사가 있고, 마지막 8개월 정도는 미국팀과 근무를 했어도 \
막상 말을 할 일은 거의 없다 보니 적당히 쓰고 읽는 건 해도 말하는 것에 부담이 컸다.

더군다나 요즘은 GPT나 번역기가 충분히 잘해서, \
글을 쓸 때에는 GPT한테 검수라도 맡길 수 있었다.

말은 또 다른 문제 같았다. \
바로 듣고 바로 말을 해야 할 것 같고, \
모르는 말을 찾아보거나, 내 생각을 글로 적어보고 고칠 수도 없었다.

21살쯤 학회에서 포스터 발표를 할 때, \
이탈리아 사람의 질문이 하나도 안 들려 꽤나 애를 먹었던 경험이 있다 보니 \
더더욱 이 세미나에 대한 걱정이 컸던 것 같다.

걱정만 하면 뭐 하나. \
아직 벌어놓은 돈도 여유가 있던 시기기에 \
학원 갈 시간은 없겠다, 바로 영어 어플 두 개 1년권을 질렀다.

하나는 말해보카이고, 하나는 스픽이다. \
우선 생활 영어 단어에 내가 생각보다 문외한이란 생각이 들었고, \
말하고 듣는 것도 해봐야 트인다고, 뒤 돌아보지 않고 긁어두었다. 

둘 다 5월 발표에서 뭔가 큰 도움을 주진 않았다. \
그냥저냥 원래 실력으로 어떻게든 넘어갔던 것 같다. 

스픽은 그렇게 한 두어 달 하다가 어느샌가 안 하게 되었고, \
말해보카는 출근길에 30분씩 거의 1년째 이어오고 있다.

말은 스픽 보다는 연구실의 프랑스인 친구와 이야기하면서 더 트인 것 같고, \
생각보다 스픽에서 배운 어휘를 생활 중 쓸 만큼 내가 말이 많은 편은 아니구나 싶었다. 

말해보카는 그래도 도움이 많이 되었다. \
괜히 아침에 봤던 단어가 어느 날 미드 쇼츠에서 보이면 반갑기도 하다. \
아 이건 진짜 10만 원 값 했단 생각도 든다. 

지금은 그래도 부담은 많이 줄어들었다. \
한국어만큼 뉘앙스나 톤을 전달할 수준은 아니지만, \
온몸 비틀기 하면 뭘 말하려 했는지 정도는 전달되지 않을까 싶다. \
내가 못 알아들으면 상대방이 포기하거나 순화하겠지 싶은 뻔뻔함만 늘었다.

**5월**\
**통원**

3월 이사를 오고 나니, 원래 다니던 병원에 다닐 시간이 애매해졌다. \
왕복 2시간이 넘는 거리기에, 다녀오거나 시간에 맞춰 가기 어려웠다. \
병원에서는 약 먹고 괜찮으면 안 와도 된다고 말했지만, \
환경이 바뀌고 스트레스가 심해지니 증상이 재발했다. 

우선은 근처 병원을 알아봤다. \
문제는 근처의 거의 모든 병원이 지금 예약하면 한 달 뒤에야 방문이 가능하다는 것이다.

울며 겨자 먹기로 원래 병원으로 통원하였고, \
이후 운이 좋게 근처 병원에 자리가 생겨, 진단서를 끊어 전원을 했다.

다행히 첫 학기 연구실 생활 자체는 금방 적응해 나갔다. \
1주일에 한 번 연구 보고에, 연구실 분위기도 첫 학기부터 학생을 밀어붙이지는 않았고, \
해야 할 일도 연구 기반 조사와 코스웍 정도이니 크게 스트레스받을 일도 없었다. 

스트레스가 줄고 나니 증상도 다시 완화되었고, \
의사 선생님과 이야기 나눠 다시 약을 줄여나갔다.

완치가 없다지만, 괜찮다가도 이따금 그럴 때면 까마득하기만 하다. \
적응이 된 것 같다가도 이게 참 쉽지 않다.

**6월**\
**부산 여행**

{{< figure src="/images/post/on2025/pusan.png" width="80%" caption="25.06.13.부산" >}}

올해도 빠질 수 없는 부산 여행이다. \
매번 가는 곳이 같다 보니 이젠 친구들이 사진만 봐도 얘 또 부산 갔네 한다. \
이렇게 보니 사진도 매번 같은 구도인 것 같다.

올해도 특별한 건 없었다. \
매번 가는 호텔에, 매번 가는 이자카야, 장어덮밥, 바닷가에서의 맥주가 다였다. 

한 가지 달랐던 것은 근처에 헬스장에서 일일권 끊고 운동도 하고 왔다. \
찾아보니 옆 호텔 꼭대기 층에 헬스장이 있었다. \
뷰도 좋다고 하고, 가격도 생각보다 저렴해 방문했고, \
밑에 온천 사우나까지 연결되어 있어 피로까지 풀고 왔다.

아쉬운 건 그날 밤에 비가 와 바닷가에서 산책은 못 하고 돌아왔다. \
그래도 이렇게 한 번씩 가서 숨 돌리고 오는 게 좋다.

**7월**\
**조교**

대학원을 졸업하려면 꼭 1회는 조교 활동을 해야 한다고 한다. \
교수님께서는 70인 대형 강의를 하나 맡고 계셨는데, \
여기에 3명의 학생이 조교로 참여해야 한다고 한다.

입학한 지 2주 차, 교수님께서는 본인 강의에 조교를 해줄 수 있겠냐 물어보셨다. \
문제는 70명 강의에 졸업 조건으로 인정되는 조교는 최대 2인이었고, \
이미 졸업해야 하는 학생이 둘 있어, 나는 졸업 조건으로 인정되지는 않는다는 것이다. \
대신 장학금이 있는데 도와줄 수 있냐는 것이었다.

좋은 게 좋은 거라고 하겠다고 했다. \
이번에 한 번 해두면, 다음에 졸업 요건으로 인정되는 조교는 좀 편하게 하겠지 싶었다. 

생각보다 조교가 할 일이 적지 않았다. \
수업 자료 제작 지원, 수업 후 온라인 질의응답 게시판 대응, \
과제 및 중간, 기말고사 출제 지원, 채점 지원, 이의제기 대응, \
최종 성적산출 지원 및 출결 처리까지 \
근 80명에 왜 겨우 조교 3명밖에 지원해주지 않는지 의문이었다. 

그래도 3명이니까.. 3등분 하면 할만하겠지 싶었다. \
중간고사까지는 괜찮았다. \
함께하는 두 사람 모두 본인의 역할을 충분히 해주었다. \
문제는 이후였다. 

조교 한 명이 중간 이후부터 연락이 되질 않았고, 이후 학교에서 종적을 감추었다. \
명확한 이유는 알 수 없었고, \
증발하듯 사라진 그는 이후 수학을 멈추고 모국으로 돌아갔단 이야기만 들려왔다. \
이렇게 갑작스러울 수가 없었다. 

78명을 조교 둘이 관리해야 하는데, \
심지어 나는 졸업요건으로 인정도 받지 못하였다. \
중간 전 과제 담당이 나, 후 과제 담당이 해당 학생이었는데, \
얼떨결에 중간 후 과제도 나와 남은 조교가 담당하게 되었다. 

실로 안타까움을 금치 못하였고, 참다 참다 교수님과 행정실에 연락하였다. \
이런 상황이라면 졸업 요건 조교로 인정해 줄 수 있는지, 행정적으로 가능한지 물었다. \
다행히 교수님과 행정실 모두에서 이를 참작해 주셨고, \
나는 공식적으로 조교 졸업 요건 충족으로 인정되었다. 

억울함은 줄었지만, 이 로드를 쳐내는 건 다른 문제였다. \
내가 듣는 코스웍 과제/시험도 있고, 내 연구는 내가 챙겨야 하고, 조교 활동도 해야 한다.

난 대학원 오면 회사 다닐 때 보단 쉬엄쉬엄 다니겠지 싶었다. \
이쯤 되니 팀장직을 하던 시기의 로드와 엇비슷하다. \
석사 학위의 비용이 늘어나는 순간이었다.  

그래도 잡무 치던 짬이 있는지, 어찌어찌 학기가 마무리되었다. 

조교 중간에 기억에 남는 건 생각보다 학생들이 공부를 안 한다는 것이었고, \
그런 학생들이 GPT를 사용해 과제는 만점이지만, 시험은 저점인 경우가 잦다는 것이다. \
심지어는 GPT에 과제 문제지를 그대로 넣고 답변을 수정 없이 제출한 학생 중 일부는 \
GPT 답변이 변수명까지 같아 표절 의심으로 윤리 위원회에 회부될 뻔했다는 것이다. 

또 글씨 못 쓰는 사람이 정말 많구나 싶었다. \
답변이야 못 알아보면 긋기도 하고, 뉘앙스를 보고 부분 점수를 주기도 하지만, \
학번 이름을 못 알아보면 이게 참 답이 없다. \
나중에는 시험지 점수 엑셀에 옮기고, \
빈자리의 학생과 이름으로 추정되는 그림을 대조하여 주인을 추정하는 경우도 있었다.

정말 극한 직업이다. \
이렇게 보면 회사가 참 편하다. 

---

일처리가 괜찮았는지, 기업 특강 조교로도 몇 번 추천받아 활동하였다. \
생각보다 수입이 나쁘지 않다. 

문제는 기업 특강의 경우, 코스에서의 성적이 인사 고과에 반영되기도 하다 보니, \
과제나 시험, 성적에 더 민감해 보였다. \
본인에게 불리한 상황에 대해 더 적극적으로 컴플레인해 왔고, \
심지어는 조교를 패싱하고 책임자에게 메일을 넣는 등 다양하게 사람을 곤란하게 했다. \
회사 업무도 이렇게 하시나 궁금해지는 순간이었다.

책임자는 좋게 넘어가자는 편이었고, \
돌고 돌아 성낸 놈이 이득을 보게 되었다. \
새삼 세상엔 정말 다양한 사람이 있다. \
겪고 나니 나는 회사로 돌아가도 CS에는 못 가겠다 싶었다.

그래도 굉장히 잘 따라와 주시는 분들도 많았고, 실무 적용에 고민하시는 분들도 계셨다. \
그런 분들에게는 경험을 토대로 이런저런 방향성이나 기회를 소개해드리려 했던 것 같다.

할 때는 정말 힘들었는데, \
그래도 금융치료에 성공했다는 점에서 후회가 남진 않았다. 

---

**과제**

슬슬 나에게도 연구 과제 참여에 관한 이야기가 들려오기 시작했다. \
1년은 봐줄줄 알았는데, 입학 전부터 쓴 사람도 있다니까 

첫 창업 초창기에는 회사에서 기관 과제도 해왔다. \
다만, 내가 직접 관리는 하지 않았다 보니 실질적 참여는 처음에 가까웠다.

1년간 합류한 과제가 두 개, 내가 제안서를 쓴 과제가 두 개이다. \
중간에 참여한 과제는 현재 연차 연구 주제가 내 주제와 사실상 같아 합류하게 되었고, \
제안서를 쓴 과제는 GPU 임차 사업 지원서였다. 

그래도 봐오던 게 있어서 그런지, 그렇게 새롭지는 않았다. \
그럭저럭 현재 연차 계획도 슥슥 적어서 내고, \
제안서도 연구 주제에서 월단위 GPU 필요량 수치 근거 산정하여 작성하였다. \
그래도 회사에서 GPU 사달라고 기획서 써서 보고한 경험이 있어서 그런지 무난했다. \
운이 좋게도 작성한 과제 둘 모두 통과되었다. 

욕심은 연구에 더 집중하고 싶었던 것 같다. \
회사에서는 본인 연봉에 2~3배 정도 가치 창출하는걸 1차 목표로 하라 했는데, \
우선 그 허들은 넘어선 것 같았다. \
그러다 보니, 무언가 더 요구되는 것들에 대해서는 회의적인 시각을 가지게 되는 것 같다.

제안서를 작성하는 노고도 적지 않고, \
운이 나쁘게 마감 며칠 전에 발견하면 밤샘은 확정이었다. \
GPU를 받아오면 자연스레 관리자는 또 제안서를 쓴 내가 되어 있었다.

이것저것 용무를 처리하다 보면, 내 연구에 소홀해지는 게 보였다. \
또 주제 자체가 연구실 내에서 거의 혼자 하는 주제이다 보니, \
내가 소홀해지면 내 성과는 늦춰져만 갔고, \
늦춰진 만큼 먼저 어셉 된 Concurrent work에 구식 취급을 받아갔다.

항상 신입은 첫 성과에 목마르다고, \
논문 하나는 챙겨 졸업하고 싶었기에 더 아쉬움이 크게 남는 것 같다.
